# Daftar Belanja Material

Web daftar belanja material proyek berbasis React + Vite + Supabase.

## Environment Variables di Vercel

Tambahkan:

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

## Supabase Tables

Tabel yang digunakan:

- projects
- materials

Status material:
- Belanja
- Sudah datang

Kategori global:
- Struktur
- Atap
- Pintu & Jendela
- Kelistrikan
- Plumbing
- Finishing
- Kayu & Hardware
- Peralatan & Operasional
- Lain-lain
